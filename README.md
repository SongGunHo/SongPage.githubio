Portfolio
개인 프로젝트와 팀 프로젝트를 정리한 개발자 포트폴리오 사이트입니다.
백엔드 중심으로 웹 서비스 구현 경험을 정리했습니다.

🔗 배포 주소
👉 (배포했으면 URL 넣기)
👉 예: https://songgunho.github.io

🛠 기술 스택
Frontend: HTML, CSS, JavaScript

Backend: Java, Spring Boot / Spring Framework

Database: MySQL

Infra: Docker

Tools: IntelliJ, VS Code, GitHub

📂 주요 프로젝트
🚧 무작정 여행 웹 서비스 (진행 중)
Spring 기반 여행 기록 관리 서비스

회원가입/로그인 구현

Docker + MySQL 환경 구성

🎬 ChatBoard
Spring Boot 기반 영화 대본 관리 서비스

페이징 기반 리스트 조회 성능 개선

♻️ Vinllage FE
React/Next.js 기반 분리수거 안내 서비스

소셜 로그인 및 이메일 인증 구현

✨ 프로젝트 목적
실무에서 사용되는 웹 서비스 구조를 이해하고,
Spring 기반 백엔드 개발 역량 향상을 목표로 포트폴리오를 제작했습니다.

👤 개발자
