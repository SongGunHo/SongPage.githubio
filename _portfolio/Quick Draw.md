## 프로젝트 개요
- **기간:** 2025.05.24 (팀 프로젝트)
- **구분:** 코리아 아카데미 팀 프로젝트
- **인원:** 3명
- **Role:** 머신러닝 모델 학습 및 예측 로직 구현

## 주요 기능
- Quick Draw 데이터셋 기반 그림 분류
- 다양한 카테고리 이미지 인식
- 머신러닝 모델 학습 및 성능 평가
- 예측 결과 시각화

## 기술 스택
- **Language:** Python
- **Environment:** Jupyter Notebook
- **Libraries:** NumPy, Pandas
- **ML/DL:** scikit-learn, TensorFlow / PyTorch
- **Visualization:** Matplotlib, Seaborn

## 문제 해결 사례
- **데이터 전처리 최적화:** Quick Draw 원시 데이터를 모델 입력 형태로 변환하는 과정에서 노이즈 제거 및 정규화를 적용하여 학습 안정성 개선
- **모델 성능 개선:** 하이퍼파라미터 튜닝을 통해 분류 정확도를 향상
- **팀 협업 경험:** 데이터 전처리, 모델 학습, 시각화 역할을 분담하여 협업 기반으로 프로젝트 수행
